Training


Data processing for scientists and engineers using Rust, Python, Julia and R. You can choose which programming language you want to take the training. These training is available in English and Spanish.

Module 1: Programming principles

  • Introduction
  • Directory and file path management
  • Data and variable types
  • Mathematical and logical operators
  • Package and module management
  • Mathematical function
  • Standard and advanced data structures
  • Flow control and functions
  • Inputs and outputs of data
  • Developing simple applications using functions
  • Object oriented programming (OOP)

Module 2: Data processing

  • Main scientific packages
  • Import netCDF4, HDF5, GRIB2, TIFF, shape, XML, GeoJSON, Excel and CSV files
  • Exploratory data analysis
  • Data quality control
  • Export netCDF4, HDF5, GRIB2, TIFF, shape, XML, GeoJSON, Excel and CSV files

Module 3: Data visualization

  • Time series: curves, histograms, box plots and scatter plots
  • Maps 2D: multi-layer raster and vector data
  • Maps 3D: multi-layer raster and vector data

Module 4: Application of experimental design analysis

  • ANOVA experimental design analysis
  • Completely randomized experimental design analysis (DCA)
  • Randomized Block Design Analysis (DBA)
  • Factorial experimental design analysis
  • Experimental design analysis of simple linear regression model
  • Experimental design analysis of multivariate linear regression model

Module 5: Case study projects

  • Meteorology, air quality, agriculture and land use
  • Dynamics in hydrology and glaciology
  • Oceanography and fishing
Observed, satellite and forced data will be used to develop all case studies.
