Training

Data processing for scientists and engineers using Python, Rust, Julia and R. You can choose which programming language you want to take the training. These training is available in English and Spanish.

Module 1

  • Introduction
  • File management
  • Data type
  • Mathematical operators
  • Logical operators
  • Inputs and outputs of data
  • Mathematical function
  • Standard data structures
  • Advanced data structures
  • Flow control
  • Functions
  • Modules and packages
  • Object oriented programming (OOP)

Module 2

  • Main scientific packages
  • Import Excel and CSV files
  • Exploratory data analysis
  • Data quality control
  • Export Excel and CSV files
  • Plot time series, raster and vectorial data
  • Plot 3D graph
  • Applications of descriptive statistics

Module 3

  • Cases of study in meteorology, oceanography, hydrology and land use
  • Cases of study applied to agriculture, fishing, mining and air quality

Observed, satellite and forced data will be used to develop all case studies.
